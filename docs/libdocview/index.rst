.. _libdocview:

libdocview references
=====================

libdocview is the backend of Docview. It provides functionalities like managing
extensions and documentations. It also provides a basic search functionality.


When I need to use libdocview?
------------------------------

You might need to use libdocview one of the following two reasons:

* Accessing documentations programmatically
* Writing extensions for Docview
